# 🛠️ Windows Ops Toolkit

A collection of professional PowerShell scripts designed to automate system maintenance, initial setup, and performance optimization on Windows environments.

This toolkit is built for power users and IT professionals who want to maintain their systems' health and automate repetitive tasks using fast, one-liner commands.

## 🚀 How to Use

Every script in this repository can be executed directly from a PowerShell (Admin) terminal without the need to download the entire repo. Simply copy the command from the specific category folders.

## 📂 Project Structure

- **`maintenance/`**: Scripts for system repair (DISM, SFC), disk health, and cleaning.
- **`setup/`**: Automation for initial environment setup, folder structures, and app installations.
